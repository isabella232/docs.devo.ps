Rackspace

Rackspace is a managed cloud computing company base in Texas with datacenters in Texas, Illinois, Virginia, the United Kingdom, Australia, and Hong Kong.

Don't have a Rackspace account yet? Sign up for an account.

Example

The following node file will create a 512MB Standard Instance (size: 2) on Rackspace in their Dallas datacenter (location: dfw):

id: rackspace_server
name: Rackspace server
type: server

provider:
  name: rackspace
  location: dfw
  size: 2

Default values

This provider has no default values: you must specify size, location and image attributes.
