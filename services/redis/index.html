Redis Official documentation

Commands

start
Start Redis if stopped
stop
Stop Redis if started
reload
Reload Redis, reload the configuration and perform a graceful restart
restart
Restart Redis, reload the configuration (but kills existing connection)

Configuration

string aom.aof_rewrite_min_size Optional, Default: 64mb
Minimum size of the AOF to consider auto rewrite of the AOF based on the percentage
integer aom.aof_rewrite_percentage Optional, Default: 100
Automatically trigger AOF rewrite if the AOF grows by more of the specified percentage (0 to disable)
string aom.enable Optional, Default: no
Enable Append Only Mode
string aom.filename Optional, Default: appendonly.aof
Name of the append only file
string aom.fsync Optional, Default: everysec
Fsync mode to sync the append only file
string aom.no_appendfsync_on_rewrite Optional, Default: no
Define whether to (not) append fsync operations if another fsync op is already in progress
string bind Optional, Default: 127.0.0.1
Binding IP address.
integer databases Optional, Default: 1
Number of databases within the redis server
integer limits.maxclients Optional, Default: 10000
Max number of clients connected
integer limits.maxmemory Optional
Maximum RAM used by Redis to store data (no limit by default) in bytes
string limits.policy Optional, Default: volatile-lru
Define how Redis will remove keys when maxmemory is reached
integer limits.samples Optional, Default: 3
Sample size to use to base the policy eviction for LRU and TTL policy algorithms
string logfile Optional, Default: /var/log/redis/redis-server.log
Server log file
string loglevel Optional, Default: notice
Server log verbosity
integer lua.time_limit Optional, Default: 5000
Time limit (in ms) for the execution of a LUA script
integer port Optional, Default: 6379
Connection port.
string replication.is_slave Optional
Is this redis instance a slave?
string replication.masterauth Optional
Redis master auth password (if required)
string replication.masterip Optional
IP or host of the master server
integer replication.masterport Optional, Default: 6379
Redis port of the master server
integer replication.ping_slave_period Optional, Default: 10
Frequency (in sec) to ping the master server as health check
integer replication.priority Optional, Default: 100
Slave priority
string replication.read_only Optional, Default: yes
Make the slave server read-only (preferred option)
string replication.serve_stale_data Optional, Default: yes
Slave redis serves stale data if the connection is lost with the master
integer replication.timeout Optional, Default: 60
Timeout to Bulk IO transfer and ping response to mark the master as unreachable
string security.requirepass Optional
Require a password to connect (either client or slave)
integer slowlog.log_slower_than Optional, Default: 10000
Execution time in microsecond to catpure the slow logs; negative value to disable; 0 to log all queries
integer slowlog.max_len Optional, Default: 128
Length of the slow log to capture (FIFO based)
string snapshotting.dbfilename Optional, Default: dump.rdb
Filename where to dump the DB
string snapshotting.dir Optional, Default: /var/lib/redis
Folder where to store the snapshots
string snapshotting.rdbchecksum Optional, Default: yes
Enable database checksum (use ~10% extra CPU)
string snapshotting.rdbcompression Optional, Default: yes
Enable database compression
array snapshotting.save Optional, Default: [object Object],[object Object],[object Object]
Array of save rules including seconds and number of changes that trigger a save
string syslog.enable Optional, Default: no
Send log to syslog
string syslog.facility Optional, Default: local0
Syslog facility
string syslog.ident Optional, Default: redis
Syslog identity
integer timeout Optional
Close the connection after a client is idle for N seconds (0 to disable)
