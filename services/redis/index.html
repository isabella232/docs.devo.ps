Redis Official documentation

Commands

devops Redis start
Start Redis if stopped
devops Redis stop
Stop Redis if started
devops Redis reload
Reload Redis, reload the configuration and perform a graceful restart
devops Redis restart
Restart Redis, reload the configuration (but kills existing connection)

Configuration

aom
aom.aof_rewrite_min_size string aof_rewrite_min_size
Minimum size of the AOF to consider auto rewrite of the AOF based on the percentage Default: 64mb
aom.aof_rewrite_percentage integer aof_rewrite_percentage
Automatically trigger AOF rewrite if the AOF grows by more of the specified percentage (0 to disable) Default: 100
aom.enable string enable
Enable Append Only Mode Default: no
aom.filename string filename
Name of the append only file Default: appendonly.aof
aom.fsync string fsync
Fsync mode to sync the append only file Default: everysec
aom.no_appendfsync_on_rewrite string no_appendfsync_on_rewrite
Define whether to (not) append fsync operations if another fsync op is already in progress Default: no
bind string bind
Binding IP address. Default: 127.0.0.1
databases integer databases
Number of databases within the redis server Default: 1
limits
limits.maxclients integer maxclients
Max number of clients connected Default: 10000
limits.maxmemory integer maxmemory
Maximum RAM used by Redis to store data (no limit by default) in bytes
limits.policy string policy
Define how Redis will remove keys when maxmemory is reached Default: volatile-lru
limits.samples integer samples
Sample size to use to base the policy eviction for LRU and TTL policy algorithms Default: 3
logfile string logfile
Server log file Default: /var/log/redis/redis-server.log
loglevel string loglevel
Server log verbosity Default: notice
lua
lua.time_limit integer time_limit
Time limit (in ms) for the execution of a LUA script Default: 5000
port integer port
Connection port. Default: 6379
replication
replication.is_slave string is_slave
Is this redis instance a slave?
replication.masterauth string masterauth
Redis master auth password (if required)
replication.masterip string masterip
IP or host of the master server
replication.masterport integer masterport
Redis port of the master server Default: 6379
replication.ping_slave_period integer ping_slave_period
Frequency (in sec) to ping the master server as health check Default: 10
replication.priority integer priority
Slave priority Default: 100
replication.read_only string read_only
Make the slave server read-only (preferred option) Default: yes
replication.serve_stale_data string serve_stale_data
Slave redis serves stale data if the connection is lost with the master Default: yes
replication.timeout integer timeout
Timeout to Bulk IO transfer and ping response to mark the master as unreachable Default: 60
security
security.requirepass string requirepass
Require a password to connect (either client or slave)
slowlog
slowlog.log_slower_than integer log_slower_than
Execution time in microsecond to catpure the slow logs; negative value to disable; 0 to log all queries Default: 10000
slowlog.max_len integer max_len
Length of the slow log to capture (FIFO based) Default: 128
snapshotting
snapshotting.dbfilename string dbfilename
Filename where to dump the DB Default: dump.rdb
snapshotting.dir string dir
Folder where to store the snapshots Default: /var/lib/redis
snapshotting.rdbchecksum string rdbchecksum
Enable database checksum (use ~10% extra CPU) Default: yes
snapshotting.rdbcompression string rdbcompression
Enable database compression Default: yes
snapshotting.save array save
Array of save rules including seconds and number of changes that trigger a save Default: [object Object],[object Object],[object Object]
syslog
syslog.enable string enable
Send log to syslog Default: no
syslog.facility string facility
Syslog facility Default: local0
syslog.ident string ident
Syslog identity Default: redis
timeout integer timeout
Close the connection after a client is idle for N seconds (0 to disable)
