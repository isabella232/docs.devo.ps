MongoDB Official documentation

Commands

start
Start MongoDB if stopped
stop
Stop MongoDB if started
reload
Reload MongoDB, reload the configuration and perform a graceful restart
restart
Restart MongoDB, reload the configuration (but kills existing connection)

Configuration

bindip string bindip
Binding IP address. Default: 127.0.0.1
dbpath string dbpath
Database files path. Default: /var/lib/mongodb
enable string enable
Enable or disable mongodb service. Default: yes
enable_rest boolean enable_rest
Enable REST over the HTTP interface
logpath string logpath
Database log file path. Default: /var/log/mongodb/mongodb.log
nohttp string nohttp
Disable entirely the HTTP interface. Default: false
port integer port
Connection port. Default: 27017
user string user
Mongodb running user. Default: mongodb
