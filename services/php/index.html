PHP

PHP is a popular general-purpose scripting language that is especially suited to web development.

Fast, flexible and pragmatic, PHP powers everything from your blog to the most popular websites in the world.

Example

services:
  php: '*'
configuration:
  php:
    memory_limit: 64M
    post_max_size: 32M
    upload_max_filesize: 32M
    apc:
      shm_size: 256M

Install PHP on the node, setting the max limit of a php process to 64MB, allowing 32MB files upload, and giving 256MB of room to APC cache.

Configuration

  • apc

    • apc.enable_cli

      Enable / disable APC for the CLI version of PHP (Mostly for testing and debugging).

      Type Default Required
      integer 0 false

    • apc.enabled

      Enable / disable APC (0 - disabled).

      Type Default Required
      integer 1 false

    • apc.gc_ttl

      The number of seconds that a cache entry may remain on the garbage-collection list.

      Type Default Required
      integer 3600 false

    • apc.max_file_size

      Cache files up to this file size. Prevents large files from being cached.

      Type Default Required
      string 1M false

    • apc.shm_segments

      The number of shared memory segments to allocate for the compiler cache.

      Type Default Required
      integer 1 false

    • apc.shm_size

      The size of each shared memory segment in MB. (add unit)

      Type Default Required
      string 128M false

    • apc.ttl

      The number of seconds a cache entry is allowed to idle in a slot in case this cache entry slot is needed by another entry.

      Type Default Required
      integer 14400 false

    • apc.user_ttl

      The number of seconds a user cache entry is allowed to idle in a slot in case this cache entry slot is needed by another entry.

      Type Default Required
      integer 7200 false

  • cgi

    • cgi.fix_pathinfo

      integer 1 false

    Define whether PHP will output errors, notices and warnings.

    Type Default Required
    string Off false

    Log errors to specified file (or syslog).

    Define which errors, warnings and notices to report.

    Type Default Required

  • expose_php

    Decides whether PHP may expose the fact that it is installed on the server.

    Type Default Required
    string Off false

  • extensions

    List of extra php extensions installable via the regular OS package manager

    Type Default Required
    array false
