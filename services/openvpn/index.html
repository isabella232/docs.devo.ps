OpenVPN Official documentation

Commands

start
Start OpenVPN if stopped
stop
Stop OpenVPN if started
reload
Reload OpenVPN, reload the configuration and perform a graceful restart
restart
Restart OpenVPN, reload the configuration (but kills existing connection)
user add
Add OpenVPN client users and send credentials by email

Options

array users
list of user objects

user Object

string email
The email address to send the credentials to Valid values: email address
string name
The client name (ex. bob, alice)

Configuration

cipher string cipher
cipher Default: BF-CBC
dev string dev
Device type Default: tun
dns1 string dns1
DNS to propagate to the clients (default to OpenDNS) Default: 208.67.222.222
dns2 string dns2
DNS to propagate to the clients (default to OpenDNS) Default: 208.67.220.220
group string group
Running group of openvpn (must exist) Default: nogroup
key
key.ca_expire integer ca_expire
CA validity period (in days) Default: 3650
key.city string city
Key city Default: SanFrancisco
key.common_name string common_name
Key Common Name (CN) Default: server
key.country string country
Key country Default: US
key.email string email
Key email Default: admin@server
key.key_expire integer key_expire
Key validity period (in days) Default: 3650
key.key_size integer key_size
Increase key size if you are paranoid; it will slow down TLS negociation Default: 1024
key.name string name
Key name Default: server
key.org string org
Key Organization Default: Fort-Funston
key.org_unit string org_unit
Key Organization Unit (OU) Default: server
key.province string province
Key province Default: CA
max_client integer max_client
Maximum number of simultaneous clients Default: 100
netmask string netmask
Netmask to apply on IP range for clients Default: 255.255.255.0
network string network
Network IP range for clients Default: 10.8.0.0
port integer port
Listening port Default: 1194
proto string proto
Protocol Default: udp
user string user
Running user of openvpn (must exist) Default: nobody
