Nginx Official documentation

Commands

start
Start Nginx if stopped
stop
Stop Nginx if started
reload
Reload Nginx, reload the configuration and perform a graceful restart
restart
Restart Nginx, reload the configuration (but kills existing connection)
vhost add
Defines a HTTP virtual host in Nginx config

Options

string aliases Optional
space separated list of domain name aliases
string domain
domain name
integer port
listening port
array routes
list of route objects

route Object

string custom Optional
Custom inline nginx config to include within the route (e.g. auth, custom timeout)
string static Optional
For type static only, define how to consider the source folder - alias or root Valid values: Either of alias / root Default: root
string to
Either an upstream name, or a service / url, or a path
string type
The type of handler for that route Valid values: Either of proxy / fastcgi / websocket / uwsgi / static
string uri
Any string / regex that nginx understand as a `location` Valid values: Any string including regex
array upstreams Optional
list of upstream objects

upstream Object

array backends
List of backends associated with the upstream
string name
Name of the upstream - it must be unique on the entire node
string webroot Optional
subfolder to serve data from based on the root /var/www/_domain_

Configuration

events
events.worker_connections integer worker_connections
Number of concurrent http connection handled per nginx process Default: 1024
http
http.access_log string access_log
Global access log file path Default: /var/log/nginx/access.log
http.error_log string error_log
Global error log file path Default: /var/log/nginx/error.log
http.gzip string gzip
Enable gzip compression Default: on
http.gzip_disable string gzip_disable
Space separated list of browser to disable gzip compression for Default: msie6
http.keepalive_timeout integer keepalive_timeout
Keepalive timeout (sec) - 0 to disable Default: 65
http.sendfile string sendfile
Enable sendfile Default: on
http.tcp_nodelay string tcp_nodelay
Enable tcp_nodelay Default: on
http.tcp_nopush string tcp_nopush
Enable tcp_nopush Default: on
http.types_hash_max_size integer types_hash_max_size
Maximum size of hash tables Default: 2048
pid string pid
Pid file path Default: /var/run/nginx.pid
user string user
Nginx running user. Default: www-data
worker_processes integer worker_processes
Number of Nginx processes Default: 4
