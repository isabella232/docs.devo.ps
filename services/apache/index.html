Apache

Example

services:
  apache: '*'
configuration:
  apache:
    timeout: 60
    prefork:
      maxclients: 20
      maxrequestsperchild: 1000

Install Apache on the node, sets a timeout of 60 seconds on receive / send, set the prefork MPM to start at most 20 Apache child processes and ensure each Apache child process will handle at most 1000 requests before respawning.

Tasks

Task Options Description
start - Start sthe Apache service, it does not do anything if it is already running.
stop - Stops the Apache service, it does not do anything if it is already stopped.
restart - Restarts the Apache service, killing existing HTTP connections. Use reload to avoid killing existing connections.
reload - Gracefully restarts the Apache service, reloading the configuration.
add vhost Object Adds a virtual host to the Apache configuration.

Configuration

  • event

    • event.maxclients

      Maximum number of simultaneous client connections

      Type Default Required
      integer 150 false

    • event.maxrequestsperchild

      Maximum number of requests a server process serves.

      Type Default Required
      integer 0 false

    • event.maxsparethreads

      Maximum number of worker threads which are kept spare.

      Type Default Required
      integer 75 false

    • event.minsparethreads

      Minimum number of worker threads which are kept spare,

      Type Default Required
      integer 25 false

    • event.startservers

      Initial number of server processes to start.

      Type Default Required
      integer 5 false

    • event.threadlimit

      ThreadsPerChild can be changed to this maximum value during a graceful restart. ThreadLimit can only be changed by stopping and starting Apache.

      Type Default Required
      integer 64 false

    • event.threadsperchild

      Constant number of worker threads in each server process.

      Type Default Required
      integer 25 false

  • group

    Apache running group.

    Type Default Required
    string www-data false

  • keepalive

    Whether or not to allow persistent connections.

    Type Default Required
    string true false

  • keepalivetimout

    Number of seconds to wait for the next request from the same client on the same connection.

    Type Default Required
    integer 5 false

  • maxkeepaliverequests

    The maximum number of requests to allow during a persistent connection.

    Type Default Required
    integer 100 false

  • prefork

    • prefork.maxclients

      Maximum number of server processes allowed to start.

      Type Default Required
      integer 150 false

    • prefork.maxrequestsperchild

      Maximum number of requests a server process serves.

      Type Default Required
      integer 4000 false

    • prefork.maxspareservers

      Maximum number of server processes which are kept spare.

      Type Default Required
      integer 10 false

    • prefork.minspareservers

      Minimum number of server processes which are kept spare.

      Type Default Required
      integer 2 false

    • prefork.startservers

      Number of server processes to start.

      Type Default Required
      integer 5 false

  • timeout

    The number of seconds before receives and sends time out.

    Type Default Required
    integer 300 false

  • user

    Apache running user.

    Type Default Required
    string www-data false
