Redis

Redis is an open source, BSD licensed, advanced key-value store. It is often referred to as a data structure server since keys can contain strings, hashes, lists, sets and sorted sets.

Example

services:
  redis: '*'
configuration:
  redis:
    port: 16379
    bind: 0.0.0.0

Install Redis on the node, making it listen on all interfaces (0.0.0.0) and on port TCP/16379 instead of TCP/6379 default.

Configuration

  • port

    Connection port.

    Type Default Required
    integer 6379 false

  • bind

    Binding IP address.

    Type Default Required
    string 127.0.0.1 false

  • timeout

    Close the connection after a client is idle for N seconds (0 to disable)

    Type Default Required
    integer 0 false

  • databases

    Number of databases within the redis server

    Type Default Required
    integer 1 false

  • loglevel

    Server log verbosity

    Type Default Required
    string notice false

  • logfile

    Server log file

    Type Default Required
    string /var/log/redis/redis-server.log false

  • syslog

    • enable (syslog.enable)

      Send log to syslog

      Type Default Required
      string no false

    • ident (syslog.ident)

      Syslog identity

      Type Default Required
      string redis false

    • facility (syslog.facility)

      Syslog facility

      Type Default Required
      string local0 false

  • snapshotting

    • save (snapshotting.save)

      Array of save rules including seconds and number of changes that trigger a save

      Type Default Required
      array [object Object],[object Object],[object Object] false

    • rdbcompression (snapshotting.rdbcompression)

      Enable database compression

      Type Default Required
      string yes false

    • rdbchecksum (snapshotting.rdbchecksum)

      Enable database checksum (use ~10% extra CPU)

      Type Default Required
      string yes false

    • dbfilename (snapshotting.dbfilename)

      Filename where to dump the DB

      Type Default Required
      string dump.rdb false

    • dir (snapshotting.dir)

      Folder where to store the snapshots

      Type Default Required
      string /var/lib/redis false

  • replication

    • is_slave (replication.is_slave)

      Is this redis instance a slave?

      Type Default Required
      string false false

    • masterip (replication.masterip)

      IP or host of the master server

      Type Default Required
      string false

    • masterport (replication.masterport)

      Redis port of the master server

      Type Default Required
      integer 6379 false

    • masterauth (replication.masterauth)

      Redis master auth password (if required)

      Type Default Required
      string [object Object] false

    • serve_stale_data (replication.serve_stale_data)

      Slave redis serves stale data if the connection is lost with the master

      Type Default Required
      string yes false

    • read_only (replication.read_only)

      Make the slave server read-only (preferred option)

      Type Default Required
      string yes false

    • ping_slave_period (replication.ping_slave_period)

      Frequency (in sec) to ping the master server as health check

      Type Default Required
      integer 10 false

    • timeout (replication.timeout)

      Timeout to Bulk IO transfer and ping response to mark the master as unreachable

      Type Default Required
      integer 60 false

    • priority (replication.priority)

      Slave priority

      Type Default Required
      integer 100 false

  • security

    • requirepass (security.requirepass)

      Require a password to connect (either client or slave)

      Type Default Required
      string [object Object] false

  • limits

    • maxclients (limits.maxclients)

      Max number of clients connected

      Type Default Required
      integer 10000 false

    • maxmemory (limits.maxmemory)

      Maximum RAM used by Redis to store data (no limit by default) in bytes

      Type Default Required
      integer [object Object] false

    • policy (limits.policy)

      Define how Redis will remove keys when maxmemory is reached

      Type Default Required
      string volatile-lru false

    • samples (limits.samples)

      Sample size to use to base the policy eviction for LRU and TTL policy algorithms

      Type Default Required
      integer 3 false

  • aom

    • enable (aom.enable)

      Enable Append Only Mode

      Type Default Required
      string no false

    • filename (aom.filename)

      Name of the append only file

      Type Default Required
      string appendonly.aof false

    • fsync (aom.fsync)

      Fsync mode to sync the append only file

      Type Default Required
      string everysec false

    • no_appendfsync_on_rewrite (aom.no_appendfsync_on_rewrite)

      Define whether to (not) append fsync operations if another fsync op is already in progress

      Type Default Required
      string no false

    • aof_rewrite_percentage (aom.aof_rewrite_percentage)

      Automatically trigger AOF rewrite if the AOF grows by more of the specified percentage (0 to disable)

      Type Default Required
      integer 100

    • aof_rewrite_min_size (aom.aof_rewrite_min_size)

      Minimum size of the AOF to consider auto rewrite of the AOF based on the percentage

      Type Default Required
      string 64mb false

  • lua

    • time_limit (lua.time_limit)

      Time limit (in ms) for the execution of a LUA script

      Type Default Required
      integer 5000

  • slowlog

    • log_slower_than (slowlog.log_slower_than)

      Execution time in microsecond to catpure the slow logs; negative value to disable; 0 to log all queries

      Type Default Required
      integer 10000 false

    • max_len (slowlog.max_len)

      Length of the slow log to capture (FIFO based)

      Type Default Required
      integer 128 false
