Nginx

High performance HTTP server.

Example

services:
  nginx: '*'
configuration:
  nginx:
    worker_processes: 1

Install Nginx on the node, but spawn 1 worker only instead of 4 (default).

Managing virtual hosts and domains is handled by dedicated tasks.

Configuration

  • user

    Nginx running user.

    Type Default Required
    string www-data false

  • worker_processes

    Number of Nginx processes

    Type Default Required
    integer 4 false

  • pid

    Pid file path

    Type Default Required
    string /var/run/nginx.pid false

  • events

    • worker_connections (events.worker_connections)

      Number of concurrent http connection handled per nginx process

      Type Default Required
      integer 1024 false

  • http

    • sendfile (http.sendfile)

      Enable sendfile

      Type Default Required
      string on false

    • tcp_nopush (http.tcp_nopush)

      Enable tcp_nopush

      Type Default Required
      string on false

    • tcp_nodelay (http.tcp_nodelay)

      Enable tcp_nodelay

      Type Default Required
      string on false

    • keepalive_timeout (http.keepalive_timeout)

      Keepalive timeout (sec) - 0 to disable

      Type Default Required
      integer 65 false

    • types_hash_max_size (http.types_hash_max_size)

      Maximum size of hash tables

      Type Default Required
      integer 2048 false

    • access_log (http.access_log)

      Global access log file path

      Type Default Required
      string /var/log/nginx/access.log false

    • error_log (http.error_log)

      Global error log file path

      Type Default Required
      string /var/log/nginx/error.log false

    • gzip (http.gzip)

      Enable gzip compression

      Type Default Required
      string on false

    • gzip_disable (http.gzip_disable)

      Space separated list of browser to disable gzip compression for

      Type Default Required
      string msie6 false
