MongoDB

MongoDB (from "humongous") is an open-source document database, and the leading NoSQL database. Written in C++.

Example

services:
  mongodb: '*'
configuration:
  mongodb:
    bindip: 0.0.0.0
    port: 27018
    enable_rest: true

Install MongoDB on the node, make the service listen on all interfaces (0.0.0.0), change the listening port to TCP/27018 (instead of 27017 by default), and enable the REST interface (allowing REST api calls against the Mongo database)

Configuration

  • user

    Mongodb running user.

    Type Default Required
    string mongodb false

  • enable

    Enable or disable mongodb service.

    Type Default Required
    string yes false

  • dbpath

    Database files path.

    Type Default Required
    string /var/lib/mongodb false

  • logpath

    Database log file path.

    Type Default Required
    string /var/log/mongodb/mongodb.log false

  • bindip

    Binding IP address.

    Type Default Required
    string 127.0.0.1 false

  • port

    Connection port.

    Type Default Required
    integer 27017 false

  • nohttp

    Disable entirely the HTTP interface.

    Type Default Required
    string false false

  • enable_rest

    Enable REST over the HTTP interface

    Type Default Required
    boolean false false
